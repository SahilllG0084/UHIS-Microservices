# 🚀 UHIS Microservices Project

## 📌 Overview
Unified Health Insurance System (UHIS) is a microservices-based application designed to process health insurance eligibility using modular and scalable architecture.

## 🛠️ Tech Stack
- Java 8
- Spring Boot 2.7
- Spring Cloud (Eureka, API Gateway)
- Feign Client
- MySQL
- JWT Security
- Swagger (OpenAPI)
- JavaMailSender
- Apache POI & OpenPDF

## 🧩 Microservices
- Service Registry (Eureka)
- API Gateway
- Admin Service
- Application Registration (AR)
- Data Collection (DC)
- Eligibility Determination (ED)
- CaseWorker Service
- Correspondence Service
- SSA Service

## 🔐 Features
- Microservices architecture
- Service discovery using Eureka
- API Gateway routing
- Inter-service communication using Feign
- JWT-based authentication (CaseWorker service)
- Email notifications (Registration, Login, Forgot Password)
- PDF & Excel report generation
- REST API documentation using Swagger

## ▶️ How to Run
1. Start Service Registry
2. Start all microservices
3. Start API Gateway
4. Access APIs via Gateway (http://localhost:8765)
